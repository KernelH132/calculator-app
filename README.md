# Scientific Calculator

A mobile Android application developed in **Kotlin** and **XML**.

## Features
* **Basic Arithmetic:** Addition, Subtraction, Multiplication, Division.
* **Scientific Operations:** Trig (`sin`, `cos`, `tan`), Hyperbolic Trig (`sinh`, `cosh`, `tanh`), Exponents, Square Root, Factorials, Permutations (`nPr`), and Combinations (`nCr`).
* **Multi-Screen Mode:** Dedicated Activity screen for calculating **Datasets (Mean, Median, Std Dev)** and **$2 \times 2$ Matrix Determinants**.

## Tech Stack
* **Language:** Kotlin
* **UI:** Traditional XML Layouts
* **Engine:** `exp4j` expression parser

## How to Run
1. Open this project in **Android Studio**.
2. Let Gradle complete the auto-sync.
3. Select an Emulator (API 24+) and click **Run**.
